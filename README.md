📦 React Fake Store App

A simple e-commerce demo built with React that uses the Fake Store API to fetch and display products. Users can browse products, view product details, add items to the cart, and see the cart count in the navigation bar.

👉 Live Demo: https://react-fakestore-app1.netlify.app/

🚀 Features

🛍 Fetches product data from the Fake Store API

🧾 Product listing page with images, titles, and prices

🔍 Product details page with description and price

🛒 Add products to a cart

🧠 Cart state managed with React Context

🛠 Built with React, React Router, Axios, and React Bootstrap
